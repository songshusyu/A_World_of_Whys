# 智喵学堂 (A World of Whys)

一个基于Spring Boot和Bootstrap的智能学习平台。

## 项目特点

- 现代化的用户界面
- 智能问答功能
- 课程管理系统
- Markdown笔记功能
- 学习进度追踪

## 技术栈

- 前端：Bootstrap 5, JavaScript
- 后端：Spring Boot, JPA
- 数据库：MySQL
- AI集成：通义千问API

## 快速开始

1. 克隆仓库
```bash
git clone git@github.com:songshusyu/A_World_of_Whys.git
```

2. 配置数据库
```sql
# 运行 init.sql 脚本创建数据库
```

3. 启动后端
```bash
java -jar zhimiao-learning-1.0.0-SNAPSHOT.jar
```

4. 启动前端
```bash
python start-server.py
```

5. 访问系统

## 开发说明

### 分支说明
- master: 主分支，稳定版本

- docs: 文档更新
- style: 代码格式调整

- test: 测试相关
- chore: 其他修改
